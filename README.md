##Problema 1:

##Problema 2:
 Excepcion: OverflowException.  
 Sucede por que la variable result no soporta un valor tan grande (1800) ya que es un byte (max 255).

 Ocurre por que estoy intentando acceder a la variable de un objeto que no existe, el segundo auto nunca se crea.

##Problema 4:
